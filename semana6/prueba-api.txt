Al enviar una petición POST a la ruta http://localhost:3000/registro con el JSON solicitado, el servidor devuelve una respuesta en 
formato JSON con un código de estado 200 OK. La respuesta incluye un objeto con tres propiedades: "estado" (con el valor "Datos recibidos"), 
"nombre" (con el valor "Maria") y "mensaje" (con el valor "Hola comunidad"). Esto confirma que el middleware express.json() 
procesó correctamente el cuerpo de la petición (req.body) y que el servidor capturó y retornó los datos estructurados de manera exitosa.

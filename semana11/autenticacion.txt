El login es el proceso mediante el cual un usuario ingresa a una aplicación utilizando sus credenciales para verificar su identidad. Esto permite que cada persona acceda únicamente a las funciones que le corresponden. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes dentro del sistema. De esta forma se evita el acceso de personas no autorizadas. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
